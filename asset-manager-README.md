# スマホ対応資産管理Webアプリ

シンプルな資産管理Webアプリです。スマホ（iOS Safari）に最適化されています。

## 特徴

- **オフライン動作**: インターネット接続不要
- **自動保存**: ブラウザ内（localStorage）に自動保存
- **スマホ最適化**: タップ編集、テンキー入力対応
- **即時計算**: 合計・構成比を自動更新
- **Markdown出力**: Claude対話用に表形式でコピー可能

## 使い方

### 1. インストール

1. `asset-manager.html` ファイルをiPhoneに転送（AirDrop、メール、クラウドストレージ等）
2. ファイルアプリで開く
3. Safariで表示される
4. 共有ボタン → 「ホーム画面に追加」でアプリ化（任意）

### 2. 基本操作

#### 金額の編集
- 金額欄をタップ
- 数値を入力（テンキーが表示されます）
- 「保存」をタップ

#### Markdown表のコピー
- 画面下部の「Markdown表コピー」ボタンをタップ
- クリップボードにコピーされます
- Claudeとの対話で貼り付けて使用できます

#### データのリセット
- 「リセット」ボタンで初期サンプルデータに戻ります

### 3. データについて

- データはブラウザ内に自動保存されます
- アプリやブラウザを閉じても保存されています
- Safari のデータを消去すると削除されます
- バックアップが必要な場合は、Markdown表をコピーしてテキストファイルとして保存してください

## 技術仕様

- シングルHTMLファイル（サーバー不要）
- localStorage でデータ永続化
- レスポンシブデザイン
- iOS Safari 最適化

## 初期データについて

HTMLファイル内の初期データはサンプルです。実際のデータは各自で入力してください。

## Markdown表形式

コピーされるMarkdown表は以下の形式です：

```markdown
|項目|金額|Au|USD|JPY|CHF|OTH|
|---|---|---|---|---|---|---|
|現物|1,000,000|Au|||||
|KFT積立|100,000|Au|||||
...
|計|2,550,000|1,100,000|650,000|350,000|300,000|150,000|
|構成比|100.0%|43.1%|25.5%|13.7%|11.8%|5.9%|
```

Claudeとの対話でこの形式をそのまま貼り付けて分析や相談ができます。
